# Project 1 - Organized Developments

A structured repository containing three different mini-projects compartmentalized into specific tasks.

## 📂 Repository Structure

### /task 1: Portfolio
Features a responsive portfolio page (`portfolio.html`) and accompanying stylesheet (`portfolio.css`). It acts as a personal online resume and showcase platform.

### /task 2: Standard Web App
Houses standard interactive web application logic featuring `app.html`, `app.css`, and a functional `app.js` file for handling dynamic application behavior.

### /task 3: Resume
Contains a dedicated modular resume layout utilizing `resume.html` and `resume.css`, perfect for professional clean-cut representations.

## 💻 Tech Stack
- **HTML5:** Core structure across all tasks.
- **CSS3:** Separate and specific styling tailored for each isolated project format.
- **Vanilla JavaScript:** Powers the functionality inside the Task 2 web app module.
